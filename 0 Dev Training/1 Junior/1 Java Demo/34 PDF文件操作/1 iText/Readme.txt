. 介绍:

  iText 是主要用于生成PDF报表的 Java组建, 还可以将xml, html 文件转化为pdf 文件.

. 使用:

  安装非常方便, 下载iText.jar 文件, 在CLASSPATH中加入iText.jar的路径, 在程序中就可以使用iText类库了.
