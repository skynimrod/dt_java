Dear Reader,

This week on XML.com we have the second part in our new "Practical
XQuery" series.  Ivelin Ivanov demonstrates how XQuery can be used
to put an HTML front end onto web services. He takes the Amazon
XML/REST web services API and uses some straightforward XQuery to
create a custom web site around Amazon's "ListMania" feature.  I
found this article particularly interesting, as it highlights more
than the basic document-query-functionality applications of XQuery.

<http://www.xml.com/pub/a/2003/05/14/xquery.html>

Uche Ogbuji's "Python & XML" column alights this month on the
Python wrappers for libxml--Daniel Veillard's excellent XML parsing
library. As Uche writes, "libxml and libxslt are popular because
of their speed, active development, and coverage of many XML
specifications, with close attention to conformance."

<http://www.xml.com/pub/a/2003/05/14/py-xml.html>

Finally this week, we have an introduction to SALT, the Speech
Application Language Tags specification. SALT is part of the
emerging XML infrastructure for multi-modal user interfaces.  In
"Adding SALT to HTML" Simon Tang explains how to install
Microsoft's SALT SDK, and how to create simple web pages that
use speech as well as HTML.

<http://www.xml.com/pub/a/2003/05/14/salt.html>


Thanks for reading!


Edd Dumbill
edd@XML.com
Managing Editor, XML.com

Using Python, Jython, and Lucene to Search Outlook Email

he put together Python, Jython and Lucene in order to create a
local web service that indexed his Microsoft Outlook mail store.

http://webservices.xml.com/pub/a/ws/2003/05/13/email.html

Web Services Security, Part 3
This article discusses XML-based authentication and the sharing
of authentication information across different applications,
known as Single Sign-on. The Security Assertions Markup Language
(SAML) from OASIS provides expression in XML of authentication
information.
